# Customer Churn Analysis

## Project Overview
This project uses Exploratory Data Analysis (EDA) to identify patterns and factors that influence customer churn.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Customer Churn Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Customer Churn Distribution

## Key Insights
- Identified important factors affecting customer churn.
- Compared churn across different customer groups.
- Visualized trends using charts and graphs.

- Customer Churn Analysis.ipynb
